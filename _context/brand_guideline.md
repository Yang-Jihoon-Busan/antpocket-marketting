## 5. 타이포그래피

| 용도 | 폰트 |
| --- | --- |
| 로고/워드마크 | **Audiowide** |
| 본문 (영문) | **Lexend Deca** |
| 본문 (한글) | **Noto Sans KR** |
| 폴백 | sans-serif |

```css
/* 본문 기본 */
font-family: 'Lexend Deca', 'Noto Sans KR', sans-serif;

/* 로고 */
font-family: 'Audiowide';
```

### 운영 원칙
- 한글은 Noto Sans KR, 영문/숫자는 Lexend Deca로 가독성 우선
- 서브 헤딩(h3)은 `font-weight: 400`의 가벼운 무게 사용
- 모바일(≤768px)에서 헤딩 크기 축소(h1 1.4em, h3 1em)

## 6. UI / 레이아웃 원칙

- **프레임워크**: Bootstrap 5.3.7 기반
- 섹션은 흰색 카드(`#e0e0e0` 1px 보더) 형태로 구획
- 섹션 패딩: 데스크톱 `54px 24px` / 모바일 `32px 24px`
