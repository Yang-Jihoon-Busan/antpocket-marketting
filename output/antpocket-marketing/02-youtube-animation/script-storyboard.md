## Scene 1 — 후킹 (0~12초)

| 항목 | 내용 |
| --- | --- |
| **나레이션** | "분명히 있었는데… 창고엔 없고, 주문은 들어오고. 사장님, 오늘도 재고 때문에 진땀 빼셨죠?" |

| **키프레임 비주얼** | 구도: 가로 16:9, 중앙에 앞치마 두른 소상공인 캐릭터(플랫 일러스트)가 박스 더미 옆에서 손을 머리에 얹고 곤란한 표정. 등장요소: 쌓인 갈색 택배 박스, 한쪽에 물음표 말풍선, 우측 상단에 '주문 1건' 알림 배지. 텍스트 위치: 화면 상단 중앙에 굵은 자막 `재고, 어디 갔지?`. 배경: 옅은 아이보리 `rgb(251,252,246)`, 약간 흐릿한 채도. 분위기: 답답하지만 귀여운 톤. |

## Scene 2 — 문제 (12~26초)

| 항목 | 내용 |
| --- | --- |
| **나레이션** | "엑셀로 어떻게든 적어보지만, 수식은 깨지고 파일은 여기저기. 창고가 둘만 돼도 숫자가 안 맞기 시작하죠." |
| **화면 자막** | `엑셀 수기 관리, 이제 한계` (하단 자막 바) |
| **화면 연출/모션** | 화면 가득 찬 복잡한 엑셀 시트. 셀들이 빠르게 채워지다가 `#REF!`, `#N/A` 에러가 빨갛게 깜빡이며 번짐. 시트가 여러 개로 갈라져 화면 곳곳으로 흩어지고, 한 파일에 빨간 ✕ 표시가 떨어짐. 사장님의 한숨(작은 김 모션). 전반적으로 빨강 경고색이 잠깐 스침. |
| **키프레임 비주얼** | 구도: 16:9, 화면의 70%를 차지하는 기울어진 엑셀 스프레드시트, 셀 곳곳에 빨간 `#REF!` 오류. 등장요소: 흩어진 엑셀 파일 아이콘 3~4개(서로 화살표로 어긋나게 연결), 깨진 수식 표시, 작게 등장한 사장님이 노트북 앞에서 머리를 감싸쥠. 텍스트 위치: 하단 자막 바에 `엑셀 수기 관리, 이제 한계`. 컬러: 회색 톤 배경에 경고 빨강 포인트만. 분위기: 혼란·과부하. |

## Scene 4 — 핵심 기능 (38~58초)

| 항목 | 내용 |
| --- | --- |
| **나레이션** | "엑셀 파일 그대로 올리면 제품이 한 번에 등록되고, 입·출고는 바코드 스캔으로 끝. 제품도 창고도 거래처도 실시간으로 한눈에. 쓰던 구글시트와도 연동됩니다." |
| **화면 자막** | (순차 등장) `엑셀로 한 번에 등록` → `바코드 스캔 입·출고` → `실시간 재고 현황` → `다중 창고·거래처·구글시트 연동` |
| **화면 연출/모션** | 4분할 또는 순차 슬라이드로 기능을 빠르게 보여줌. ① 엑셀 파일 아이콘이 휴대폰/대시보드로 드래그되어 제품 카드들이 좌르륵 채워짐. ② 스마트폰이 바코드를 '삑' 스캔하면 재고 숫자가 +1/-1 카운트업·다운. ③ 막대 그래프와 재고 게이지가 실시간으로 차오름. ④ 두 개의 창고 아이콘 + 거래처 아이콘 + 구글시트 로고가 올리브 선으로 연결되어 하나의 대시보드로 모임. 각 전환은 경쾌한 슬라이드/팝 모션. |
| **키프레임 비주얼** | 구도: 16:9, 중앙에 태블릿/스마트폰 목업의 앤트포켓 대시보드, 주변에 4개 기능 아이콘이 시계방향 배치. 등장요소: (좌상) 엑셀→제품카드 변환, (우상) 스마트폰 바코드 스캔 '삑' 이펙트, (우하) 실시간 재고 막대그래프·게이지, (좌하) 창고2개·거래처·구글시트 로고가 올리브 선으로 대시보드에 연결. 아이콘은 올리브 `#827717` 통일. 텍스트 위치: 각 아이콘 아래 짧은 라벨, 하단에 현재 단계 자막. 배경: 화이트. 분위기: 빠르고 똑똑하고 정돈된 느낌. |
