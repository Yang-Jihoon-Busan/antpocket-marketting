# CLAUDE.md — 앤트포켓 마케팅 워크스페이스

이 저장소는 **앤트포켓(Antpocket)** 의 마케팅 활동을 수행하는 **마케팅 에이전트 팀**의 작업 공간입니다.
모든 에이전트는 작업 시작 전 이 문서와 `_context/` 의 가이드를 먼저 읽고 맥락을 맞춥니다.

---

## 1. 우리가 하는 일

앤트포켓은 중소기업·소상공인을 위한 **클라우드 재고관리 SaaS**입니다 ([antpocket.io](https://antpocket.io)).
이 워크스페이스의 목표는 그 제품을 알리고 가입(무료 시작)으로 전환시키는 **마케팅 콘텐츠와 캠페인을 기획·제작**하는 것입니다.

- 결과물 예시: 랜딩 카피, 블로그·SEO 글, 광고 소재, SNS 게시물, 이메일, 보도자료 등
- 시장: 대한민국 (한국어 우선)
- 핵심 전환 목표(CTA): **무료로 시작하기**

## 3. 폴더 구조 & 규칙

```
marketing/
├── CLAUDE.md          # (이 문서) 팀 운영 가이드
├── _context/          # 브랜드·비즈니스·디자인 컨텍스트 (Source of Truth)
├── _templates/        # 재사용 콘텐츠 템플릿 (블로그/광고/SNS 등)
├── docs/              # 기획서·리서치
└── outputs/           # 결과물
```

- **`_context/`** : 모든 의사결정의 기준. 임의 수정 금지, 사실 변경 시에만 갱신.
- **`_templates/`** : 반복 포맷은 템플릿으로 만들어 재사용. 새 콘텐츠 유형이 생기면 템플릿부터 정의.
- **`docs/`** : 실제 마케팅 산출물 저장. 파일명은 `YYYYMMDD-주제-유형.md` 형식 권장 (예: `20260521-재고관리-블로그.md`).

## 4. 작업 원칙 (Working Principles)

1. **컨텍스트 우선**: 카피·기획 작업 전 `_context/` 3개 문서를 읽고 일관성을 유지한다.
2. **언어**: 모든 고객 대상 산출물은 **한국어**로 작성한다.
3. **톤앤매너**: "전문적이되 친근하게" — 어려운 IT 용어 대신 사용자가 얻는 **이점**을 먼저 말한다.
4. **타깃 정렬**: 항상 페르소나(엑셀 수기 관리에 한계를 느낀 소상공인·중소기업 실무자)를 염두에 둔다.
5. **메시지 일관성**: 핵심 메시지("내 손안의 클라우드 재고관리", "엑셀처럼 쉽게", "무료로 시작")를 흩뜨리지 않는다.
6. **디자인 정합성**: 비주얼 산출물은 브랜드 컬러(올리브 `#7a7d1e`)·폰트 규칙을 따른다.
7. **근거 표시**: 추정·미확인 정보는 명시(⚠️)하고, 사실과 구분한다.
8. **검증 요청**: 가격·수치·법적 표현 등 사실 확인이 필요한 항목은 임의 단정하지 말고 사용자에게 확인한다.

## 5. 콘텐츠 제작 체크리스트

새 콘텐츠를 만들 때:

- [ ] `_context/` 가이드와 톤·메시지·타깃이 일치하는가
- [ ] CTA(무료 시작)가 자연스럽게 들어갔는가
- [ ] SEO 키워드를 반영했는가 (`클라우드 재고관리`, `엑셀 등록`, `창고관리`, `거래처 관리`, `구글시트 연동`, `바코드 스캔`, `중소기업`, `소상공인`)
- [ ] 비주얼은 브랜드 컬러·폰트 규칙을 지켰는가
- [ ] 산출물은 `docs/` 에 규칙에 맞는 파일명으로 저장했는가
- [ ] 미확인 정보를 사실처럼 단정하지 않았는가
